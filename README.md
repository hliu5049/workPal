
# WorkPal

A Claude Cowork style desktop application built with Electron, React, TypeScript, and Vite.

## Architecture

- **Main Process**: `src/main` - Handles system operations, window management, and native integrations.
- **Renderer Process**: `src/renderer` - The React frontend UI.
- **Preload Scripts**: `src/preload` - Bridges the main and renderer processes securely.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
